# Nuri.Duxel Navigation Sample

Run the sample with:

```powershell
dotnet run --project "samples\Duxel\Nuri.DuxelNavigationSample\Nuri.DuxelNavigationSample.csproj" -c Debug
```

The sample demonstrates platform-neutral `useNavigation` and `Router` usage through the Duxel Windows host. It covers `Navigate`, `Replace`, `GoBack`, route-local hook state, keyed route replacement, and a Duxel opacity transition when each screen enters.
