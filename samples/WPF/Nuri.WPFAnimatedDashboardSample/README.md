# Nuri WPF Animated Dashboard Sample

This WPF-only sample exercises Margin, background, foreground, and Rotate transitions.

```powershell
dotnet run --project "samples\WPF\Nuri.WPFAnimatedDashboardSample\Nuri.WPFAnimatedDashboardSample.csproj" -c Release
```

Toggle the dashboard repeatedly to replace active transitions, then confirm that each card settles on the latest values.
